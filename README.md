# Backend Labs
#### backend-labs

Enterprise backend laboratory exploring modern backend architecture and AI-powered features using .NET 8 and Angular.

## Goals

- Demonstrate scalable backend architecture (DDD / CQRS)
- Build production-style APIs with ASP.NET Core
- Explore AI integrations (Chat Completions, RAG, Agents, MCP)
- Implement shift-left testing with Playwright
- Serve as a technical showcase for senior backend engineering roles
